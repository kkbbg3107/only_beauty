Only Beauty 薪資計算系統
========================

此專案包含以下檔案：

1. salary_calculator.py - 主要的薪資計算程式
2. README.md - 詳細的使用說明文件
3. requirements.txt - Python套件依賴清單

使用方法：
1. 確保已安裝 Python 3.7+
2. 安裝必要套件：pip install -r requirements.txt
3. 執行程式：python salary_calculator.py
4. 按照提示輸入Excel檔案路徑和員工人數

程式會自動：
- 找出Excel中數字最大的工作表
- 讀取E5(總業績)和E7(總消耗)
- 計算顧問和美容師/護士的獎金分配
- 顯示詳細的計算結果

詳細說明請參考 README.md 檔案。
